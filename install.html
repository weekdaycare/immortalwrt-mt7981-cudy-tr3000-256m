准备 ​

你需要的准备的东西有：

  • 一台 PC 通过网线连到你的路由器
  • 官方过渡中间件
  • 固件包

刷入官方过渡件 ​

首先进入 Cudy 后台管理界面，192.168.10.1，完成初始设置后找到 固件升级

在这里上传 官方过渡中间件，然后确定，等待系统重启升级完成（白灯常亮表示系统启动成功）

NOTE

SN 码 2544 以后的新批次 128M 设备必须使用新的官方过渡件刷入

刷入固件包 ​

连接网线到路由器，打开浏览器，输入 192.168.1.1，进入过渡件的后台管理界面，系统 -> 备份与升级 -> 刷写新的固件

然后上传 固件包，然后确定，等待系统重启升级完成

进入后台 ​

Immortalwrt 的管理后台地址为 192.168.6.1 ，你也可以通过 http://immortalwrt.lan/ 访问
